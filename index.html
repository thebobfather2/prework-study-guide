HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The command to stage changes is: git add -A
  • To commit changes: git commit -m "commit note here" ...'-m' means "message"
  • Before opening a pull request, make sure code is up to date with: git pull origin main
  • Push changes with: git push origin feature/add-html

JavaScript

  • To create variables, use "var" then name it, and add "= []" with variables separated by commas.
  • Strings must be surrounded by quotes and each statement ends with ;
  • "for" loops using arrays are structured: for (var x = 0; x < variable.length; x++) {console.log(variable[x]);}
